ხშირად დასმული კითხვები

სტანდარტული დანერგვა 2-3 სამუშაო დღეში ხდება. ეს მოიცავს სისტემის ინსტალაციას, მონაცემების იმპორტს Excel-იდან და გუნდის საბაზისო ტრენინგს. ენტერპრაიზ ვერსიისთვის — 1-2 კვირა, რაც მოიცავს ინდივიდუალურ კონფიგურაციას.
დიახ, სისტემა მხარს უჭერს მასობრივ იმპორტს Excel ფაილებიდან. ჩვენ დაგეხმარებით მონაცემების სწორად სტრუქტურირებაში და მიგრაციაში. ვალიდაცია და დუბლიკატების ფილტრაცია ავტომატურად ხდება.
არა. AssetFlow Studio ეშვება თქვენს ლოკალურ სერვერზე (On-Premise). ინტერნეტთან კავშირი არ არის აუცილებელი — სისტემა სრულად ფუნქციონირებს შიდა ქსელში. ეს ასევე ნიშნავს, რომ თქვენი მონაცემები მხოლოდ თქვენთან რჩება.
სერვერი: Windows Server + SQL Server (ან Linux). კლიენტი: ნებისმიერი თანამედროვე ბრაუზერი (Chrome, Edge, Firefox). სპეციალური პროგრამის ინსტალაცია კლიენტის მხარეს არ არის საჭირო — ყველაფერი ბრაუზერში მუშაობს.
14 დღის საცდელი პერიოდი სრულიად უფასოა. ამ პერიოდში სისტემის ყველა ფუნქცია ხელმისაწვდომია. საცდელი პერიოდის შემდეგ თქვენ ირჩევთ შესაფერის გეგმას. თუ სისტემა არ მოგეწონათ — არანაირი ვალდებულება.
სტარტერ გეგმა 3 მომხმარებელს, პროფესიონალი — 10 მომხმარებელს მხარს უჭერს. ენტერპრაიზ ვერსია ულიმიტოა. ყველა მომხმარებელს აქვს RBAC-ით განსაზღვრული ინდივიდუალური წვდომის უფლებები.
